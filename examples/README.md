# Examples
The examples in this directory are probably the easiest way to learn how to use LabTerm. Play around!

0. **Hello, World!** shows how to place items on the dashboard.
1. **Getting Started** shows how to define, control and monitor a simple instrument.
2. **Power Control** tries to simulate a possible application of LabTerm, with several instruments and complex actions.
3. **Server Status** is a simple example of how LabTerm can be used for remote monitoring.
